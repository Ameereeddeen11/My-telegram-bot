# My-telegram-bot
## Features
- Python 3.11.6
- Python Pip 21.3.2
- Python Telegram Bot 
- Python Requests
- Open Weather API

This project is a telegram bot that can send you the weather of a city. It uses the Open Weather API to get the weather data.
It can also send who has the birthday today. It uses a json file to store the birthdays. 
I'm going to add more features such as game tic-tac-toe in the future.

## How to use
1. Clone the repository
2. Install the requirements with `pip install -r requirements.txt`
3. Create a file called `token.txt` and put your telegram bot token, API Keys in it
4. Run the `main.py` file
5. Enjoy
